Hello World!

Calum Mackervoy -- December 17, 2017 00:05

The designs complete, the project created, dependencies initialised and template established, it was time to start coding. Now that we've got some initial functionality in place, we decided that it was time to put up a "hello world" post on the website :-)

As you can see, the map is in place, pins can be added to it with arbitrary titles and descriptions, which can then be accessed in a list, edited and destroyed at the press of a few buttons.

With the goal of releasing an alpha version of the app by May next year, the next objective is to establish networking capability within the app. A secondary priority will be to begin with the styling of the app, so that it has a user-friendly, intuitive flow with a unique look & feel.
